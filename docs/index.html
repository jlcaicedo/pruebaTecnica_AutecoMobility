Prueba técnica Desarrollador React Native

Objetivo: Conocer más tus habilidades técnicas
Tiempo de la prueba: 5 días a partir de la recepción de la prueba.

La siguiente prueba consiste en el desarrollo de una aplicación App para revisar la Información de las películas más populares del momento.

Misión:

Construir una aplicación en React Native que permita:

  1. Películas Populares (Home): Se mostrará el listado de películas, debe ser mostrado en forma de grid.
  2. Detalle de Película: Se puede ver el detalle de la película y sus reviews.
  3. Mis Películas Favoritas: Puedo agregar y eliminar las películas que me gusten a mi lista de favoritos.
  4. Inicio de sesión: Inicio de sesión o crear cuenta si lo desean
  5. Cámara: Los usuarios podrá hacer uso de la cámara o rollo fotográfico del dispositivo para guardar su foto de perfil.
  6. Ubicación: La app debe solicitar permisos de ubicación del usuario.

Funcionalidades básicas:

  • Listar las películas favoritas en el home.
  • La aplicación debe permitir agregar y eliminar películas del listado de favoritas.
  • Debe haber un indicador en el home de aquellas películas que ya están en el listado de favoritas.
  • La aplicación debe permitir la navegación a las distintas páginas (Home, Detalle, Favoritos).
  • En la página de detalle de película se debe mostrar la información general y los reviews, también debe poderse agregar a favoritos.
  • Los usuarios podrán agregar y editar favoritos si tienen una sesión iniciada
  • Sesión y Favoritos haciendo uso del Local Storage.

Se evaluará en términos de:

  • Buenas prácticas de diseño en términos de apropiada separación de responsabilidades
  • entre los componentes de la solución.
  • Uso apropiado de los elementos del patrón MVC.
  • Nivel de desacoplamiento de los componentes de la solución.
  • Uso de las capacidades de React Native.
  • Claridad y legibilidad del código.
  • Manejo y control de errores.
  • Uso de elementos de ES6.
  • Manejo de los estilos de componentes.

API a utilizar:

  • GET Popular Movies
  • GET Movies Detail
  • GET Movies Reviews

Herramientas a utilizar:

  • Expo
  • React ( Punto a favor el uso de componentes funcionales y hooks )

Condiciones de entrega:

  • Se deberá entregar la solución en un repositorio de GIT público (puede ser Github, Gitlab o Cualquiera de preferencia).
  • Aplicaciones móviles compiladas para IOS y Android
  • El README.md debe contener una breve explicación de cómo correr el proyecto.
  • El entregable será una versión con el tag “v1.0”.
  • El aplicativo se deberá desplegar en EXPO canal de desarrollo “DEV”
